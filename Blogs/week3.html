Interaction Design for the web

Week 3 (26/02 - 01/03)

Interaction Design - Lecture:

This week''s lecture built onto what we learnt last week about interaction. 'Interaction'' is still pretty much similar to what we learnt last week. We learnt that something is interactive if it is responsive and doesn't respond in a completely random manner - yes, the opposite of what we learnt 'interaction' to be. We also learnt about how interaction between a user and a system is done through interface. Interface is the symbolic meeting place between an active body and that which it can affect [1]. In simple words, this means that interface is the space or environment which allows you the opportunity to interact with certain things. Interface is also described as designing a way to represent and control interactions that take place within a system [2]. This falls back onto how we would like our websites to look and how we would want them to be navigated around. This would mean that our website is sort of the interface that allows the users to interact with your content. It also reflects on how we would like the system to work for users and how the user can interact with it.

Semantic Markup:

This week we learnt about semantic markup in html. Seems like it's a very practical and structured way to make it easier to read and understand code for others looking at it, as well as being beneficial for your own understanding. It's definitely made it easier to code this website because I had a lot of unnecessary code, I didn't even understand that was incorporated into it. Building on the website, I've found it quite cool to place images and links into the code, as well as making use of hypertext throughout each of my pages. Its also pretty cool adding in these blog posts. Some of the characters I've been using in Microsoft Word when I type out my blogs, aren't recognised in html code. So, there's a lot of sifting I have to do after I've copied and pasted my blog text into visual studio. In a weird way, its so cool to see the page take shape once I've done all the editing. It makes me feel very professional and seem like I know what I'm doing with regards to building this website and filling it with content.
